How it works
01

Install on both browsers

You and your partner each add Duet to Chrome — takes about 10 seconds.

02

Create a room

One person clicks "Create a room" and shares the 6-character code.

03

Open the same video

Both navigate to the same video on YouTube, Vimeo, or any HTML5 site.

04

Press play. You're in sync.

Every pause, seek, and reaction stays in lockstep automatically.
